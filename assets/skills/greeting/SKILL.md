# 问候 Skill

## 功能
根据当前时间自动生成合适的问候语。

## 使用方法

### 自动触发
```markdown
用户：早上好
助手：早上好！祝您今天工作顺利！
```

## 规则
- 05:00-11:00: 早上好
- 11:00-13:00: 中午好
- 13:00-18:00: 下午好
- 18:00-22:00: 晚上好
- 22:00-05:00: 夜深了

## 示例
- "早上好"
- "晚上好"
- "你好"
