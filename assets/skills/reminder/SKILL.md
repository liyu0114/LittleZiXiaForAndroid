# 提醒 Skill

## 功能
设置定时提醒，到时间自动提醒。

## 使用方法

### 通过对话
```markdown
用户：5分钟后提醒我开会
助手：✅ 已设置提醒：5分钟后提醒您：开会
```

## 参数
- `time` (string): 提醒时间（5分钟后、明天早上8点等）
- `content` (string): 提醒内容

## 示例
- "5分钟后提醒我开会"
- "明天早上7点叫我起床"
- "每周一9点提醒我开会"
