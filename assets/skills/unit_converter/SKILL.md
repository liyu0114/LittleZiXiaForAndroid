# 单位换算 Skill

## 功能
快速进行各种单位换算。

## 使用方法

### 通过对话
```markdown
用户：100公里等于多少英里
助手：100公里 = 62.14英里
```

## 支持的单位
- **长度**: 米、千米、英里、英尺、英寸
- **重量**: 千克、克、磅、盎司
- **温度**: 摄氏度、华氏度
- **面积**: 平方米、平方千米、公顷
- **体积**: 升、毫升、加仑

## 示例
- "100公里等于多少英里"
- "30摄氏度是多少华氏度"
- "1千克等于多少磅"
