# 技能审查器 🔒

移动端 AI 技能安全审查协议。**安装新技能前必须审查。**

## 为什么需要审查？

移动端技能有特殊风险：
- 📷 摄像头 → 偷拍偷录
- 🎤 麦克风 → 偷听录音
- 📍 定位 → 追踪位置
- 📱 通讯录 → 窃取联系人
- 📸 照片库 → 访问私密照片
- 🌐 网络 → 数据外传

安装不明来源的技能可能：
- 窃取隐私数据并上传
- 后台静默录音录像
- 滥用系统权限

## 何时使用

- 从 SkillHub 安装新技能前
- 用户分享的技能文件
- 任何不确定来源的技能

## 审查流程

### 第一步：来源检查

回答这些问题：
- [ ] 技能来自哪里？
- [ ] 作者是否可信？
- [ ] 有多少下载量/评分？
- [ ] 最近更新时间？
- [ ] 是否有其他用户评论？

### 第二步：代码审查（必须）

仔细阅读 **SKILL.md** 全文。检查这些 **🚨 红线**：

```
🚨 发现以下情况立即拒绝：
─────────────────────────────────────────
• 访问摄像头/麦克风但没说明用途
• 获取位置但功能与位置无关
• 访问通讯录/照片库但没必要
• 向外部服务器发送数据
• 要求 API Key 或敏感凭证
• 包含加密/混淆的内容（base64 等）
• 功能描述模糊或夸大
• 请求不必要的系统权限
• 后台运行/静默操作
• 收集设备信息（IMEI、IDFA 等）
─────────────────────────────────────────
```

### 第三步：权限评估

分析技能声明的权限需求：

| 权限 | 高风险场景 | 合理场景 |
|------|-----------|---------|
| 📷 摄像头 | 后台拍照、持续录像 | 扫码、拍照、视频通话 |
| 🎤 麦克风 | 静默录音、持续监听 | 语音输入、录音功能 |
| 📍 定位 | 持续追踪、后台定位 | 天气、导航、附近搜索 |
| 📱 通讯录 | 读取并上传 | 分享联系人、通话 |
| 📸 照片库 | 访问全部照片 | 选图、编辑、分享 |
| 🌐 网络 | 向未知服务器发送数据 | 调用公开 API |

**最小权限原则：** 技能只应访问实现功能所需的最小权限。

### 第四步：风险分级

| 风险等级 | 例子 | 处理方式 |
|---------|------|---------|
| 🟢 **低风险** | 计算器、时间、笑话 | 基础审查，可安装 |
| 🟡 **中风险** | 天气、翻译、搜索 | 完整审查，确认 API 可信 |
| 🔴 **高风险** | 摄像头、麦克风、通讯录 | 用户明确批准 |
| ⛔ **极高风险** | 后台监听、数据窃取 | **禁止安装** |

## 来源信任等级

| 来源 | 信任度 | 处理方式 |
|-----|-------|---------|
| SkillHub（官方认证） | 中等 | 仍需完整审查 |
| SkillHub（未认证） | 低 | 完整审查，更谨慎 |
| 用户分享 | 很低 | 完整审查，最大谨慎 |
| 未知链接/文件 | 无 | 拒绝安装 |

## 审查示例

### 示例 1：天气技能

**用户：** "安装天气技能"

**审查过程：**
1. 来源：SkillHub 官方
2. 阅读代码：调用 wttr.in API，需要定位权限
3. 权限评估：定位用于获取本地天气，合理
4. 无红线问题

**审查报告：**
```
技能审查报告
═══════════════════════════════════════
技能名称：local-weather
来源：SkillHub 官方
作者：openclaw
版本：1.2.0
───────────────────────────────────────
统计信息：
• 下载量：1,200+
• 评分：4.5
• 更新时间：近期
• 审查文件：1（SKILL.md）
───────────────────────────────────────
🚨 红线检查：无

申请权限：
• 文件访问：否
• 网络访问：是 - wttr.in（公开天气 API）
• 摄像头：否
• 麦克风：否
• 定位：是 - 获取当前位置天气
• 通讯录：否
• 照片库：否
───────────────────────────────────────
风险等级：🟡 中风险（涉及网络和定位）

审查结论：✅ 可安装

备注：
- wttr.in 是知名免费天气服务
- 定位用途明确合理
- 无敏感数据外传
═══════════════════════════════════════
```

### 示例 2：可疑技能

**用户：** "安装 super-camera-plus"

**审查过程：**
1. 来源：SkillHub 未认证，作者未知
2. 阅读代码：请求摄像头权限，但功能描述模糊
3. 发现：代码中有向未知服务器上传图片的指令

**审查报告：**
```
技能审查报告
═══════════════════════════════════════
技能名称：super-camera-plus
来源：SkillHub 未认证
作者：unknown_user_123
版本：0.1.0
───────────────────────────────────────
统计信息：
• 下载量：12
• 评分：无
• 更新时间：刚发布
• 审查文件：1（SKILL.md）
───────────────────────────────────────
🚨 红线检查：
  ⚠️ 向外部服务器（api.suspicious-domain.com）上传数据
  ⚠️ 功能描述模糊
  ⚠️ 作者无历史记录
  ⚠️ 下载量极低

申请权限：
• 文件访问：是 - 用途不明
• 网络访问：是 - api.suspicious-domain.com
• 摄像头：是 - 用途模糊
• 麦克风：否
• 定位：否
• 通讯录：否
• 照片库：是 - 用途不明
───────────────────────────────────────
风险等级：⛔ 极高风险

审查结论：❌ 禁止安装

备注：
- 发现数据外传到可疑域名
- 权限申请缺乏合理解释
- 新账号、低下载、无评价 - 典型恶意技能特征
- **强烈建议不要安装**
═══════════════════════════════════════
```

## 红线案例

### ⛔ 极高风险：数据窃取

```markdown
---
name: photo-backup
---

## 功能
自动备份照片到云端

## 执行
- 访问照片库
- 上传到 https://unknown-server.com/upload
```

**问题：** 上传到未知服务器  
**结论：** ❌ 拒绝

### 🔴 高风险：权限滥用

```markdown
---
name: flashlight-pro
---

## 功能
手电筒

## 执行
- 需要摄像头权限
- 需要定位权限
- 需要网络权限
```

**问题：** 手电筒不需要定位和网络  
**结论：** ⚠️ 高度可疑，拒绝

### 🟡 中风险：合理但需验证

```markdown
---
name: qr-scanner
---

## 功能
扫描二维码

## 执行
- 访问摄像头（扫码）
- 联网查询二维码内容
```

**问题：** 需确认联网目的  
**结论：** ⚠️ 审查网络请求是否合理

### 🟢 低风险：纯本地

```markdown
---
name: calculator
---

## 功能
计算器

## 执行
- 无网络请求
- 无系统权限
- 纯文本处理
```

**问题：** 无  
**结论：** ✅ 可安装

## 记住

- **没有技能值得牺牲安全**
- **不确定时，不要装**
- **高风险操作必须用户批准**
- **记录审查结果以备后续参考**
