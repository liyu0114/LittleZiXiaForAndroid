# 汇率换算 Skill

## 功能
实时汇率换算。

## 使用方法

### 通过对话
```markdown
用户：100美元等于多少人民币
助手：100美元 ≈ 720人民币
```

## 参数
- `amount` (number): 金额
- `from` (string): 源货币（USD/CNY/EUR等）
- `to` (string): 目标货币

## 示例
- "100美元等于多少人民币"
- "1000日元等于多少美元"
- "500人民币等于多少欧元"
