## 二、OpenClaw 的工具系统

### 2.1 工具类型

| 类型 | 示例 | 说明 |
|------|------|------|
| 内置工具 | read/write/exec | OpenClaw 核心功能 |
| 技能 | SKILL.md | 用户自定义 |

| 节点 | camera/location | 远程设备能力 |

### 2.2 工具定义 (SKILL.md)

```markdown
---
name: my_skill
description: 技能描述
---

# 技能说明

当用户...时，执行...

## 可用工具

- `bash` - 执行命令
- `read` - 读取文件
```

### 2.3 工具调用流程

```
LLM → Tool Call → Tool Execution → Result → LLM → Response
```

## 七、关键技术点

### 7.1 WebSocket 连接

```dart
class WebSocketConnection {
  WebSocket? _socket;
  
  Future<void> connect(String url) async {
    _socket = await WebSocket.connect(url);
    _socket!.listen(_onMessage);
  }
  
  void send(Map<String, dynamic> data) {
    _socket!.add(json.encode(data));
  }
}
```

### 7.2 HTTP API 调用

```dart
class OpenAICompatibleClient {
  final String baseUrl;
  final String apiKey;
  
  Future<String> chat(String modelId, List messages) async {
    final response = await http.post(
      '$baseUrl/chat/completions',
      headers: {
        'Authorization': 'Bearer $apiKey',
        'Content-Type': 'application/json',
      },
      body: json.encode({
        'model': modelId,
        'messages': messages,
      }),
    );
    return response.body;
  }
}
```
