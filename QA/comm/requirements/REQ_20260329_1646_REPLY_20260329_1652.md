### 1. UI 适配方案

```dart
// 响应式布局
Widget build(BuildContext context) {
  final width = MediaQuery.of(context).size.width;
  
  if (width > 600) {
    // 平板：侧边栏
    return Row(children: [TopicSidebar(), ChatView()]);
  } else {
    // 手机：底部抽屉
    return Column(children: [ChatView(), TopicDrawer()]);
  }
}
```

### 2. 数据迁移

如果用户已有对话历史，需要迁移：

```dart
// 将现有对话迁移到"默认话题"
Future<void> migrateExistingChats() async {
  if (topicManager.topics.isEmpty) {
    // 创建默认话题
    final defaultTopic = topicManager.createTopic(
      title: '历史对话',
    );
    // 迁移所有历史消息
    for (final message in chatHistory) {
      defaultTopic.addMessage(message);
    }
  }
}
```
