## 技术方案

### 1. 数据模型

```dart
/// Gateway 配置
class GatewayConfig {
  final String name;
  final String host;
  final int port;
  final String token;
  final ConnectionType type;
  
  String get url => 'ws://$host:$port';
}

enum ConnectionType {
  tailscale,
  local,
  custom,
}
```

### 3. 连接测试

```dart
Future<ConnectionStatus> testConnection(GatewayConfig config) async {
  try {
    final socket = await WebSocket.connect(
      config.url,
      timeout: Duration(seconds: 5),
    );
    
    // 发送握手帧
    socket.add(jsonEncode({
      'type': 'connect',
      'params': {
        'auth': {'token': config.token},
      },
    }));
    
    // 等待响应
    final response = await socket.first.timeout(Duration(seconds: 3));
    final data = jsonDecode(response);
    
    if (data['type'] == 'hello-ok') {
      return ConnectionStatus.connected;
    }
    
    return ConnectionStatus.authFailed;
  } catch (e) {
    return ConnectionStatus.failed;
  }
}
```
