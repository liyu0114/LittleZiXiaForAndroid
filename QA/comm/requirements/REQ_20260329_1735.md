## 技术方案

### 1. 数据模型

```dart
class GatewayConfig {
  final String name;
  final String host;
  final int port;
  final String? token;
  final ConnectionType type; // tailscale / local / custom
}
```

### 2. 预设配置

```dart
class PresetGateways {
  static final List<GatewayConfig> defaults = [
    GatewayConfig(
      name: '远程龙虾',
      host: '100.80.206.8',
      port: 18789,
      type: ConnectionType.tailscale,
      token: 'QA_TOKEN', // 质检员 Token
    ),
    GatewayConfig(
      name: '本地开发',
      host: '127.0.0.1',
      port: 18789,
      type: ConnectionType.local,
      token: 'DEV_TOKEN', // 程序员 Token
    ),
  ];
}
```

### 3. 连接测试

```dart
Future<bool> testConnection(GatewayConfig config) async {
  try {
    final socket = await Socket.connect(
      config.host,
      config.port,
      timeout: Duration(seconds: 5),
    );
    await socket.close();
    return true;
  } catch (e) {
    return false;
  }
}
```
