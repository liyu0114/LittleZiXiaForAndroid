## 一、OpenClaw 话题管理研究

### OpenClaw 的会话机制

**Session Key 设计：**
```
直接消息：agent:<agentId>:<mainKey>
        或 agent:<agentId>:<channel>:dm:<peerId>（按用户隔离）
群组消息：agent:<agentId>:<channel>:group:<id>
话题线程：agent:<agentId>:<channel>:group:<id>:topic:<threadId>
```

**核心特性：**
1. **DM 隔离模式**：`dmScope: main/per-peer/per-channel-peer/per-account-channel-peer`
2. **身份链接**：同一人跨渠道共享会话
3. **会话压缩**：自动总结旧对话，保留关键信息
4. **记忆刷新**：压缩前触发静默记忆写入

**缺失：**
- ❌ 没有用户可选择的"话题"概念
- ❌ 用户无法主动创建/切换话题
- ❌ 没有话题分类标签

### 4. 话题记忆压缩（参考 OpenClaw）（2h）

```dart
/// 当话题消息数超过限制时，自动压缩
Future<void> compressTopicMemory(ConversationTopic topic) async {
  if (topic.messages.length <= _maxMessagesPerTopic) return;

  // 1. 保留最近的消息
  final recentMessages = topic.messages
      .sublist(topic.messages.length - _maxMessagesPerTopic ~/ 2);
  
  // 2. 压缩旧消息
  final oldMessages = topic.messages
      .sublist(0, topic.messages.length - _maxMessagesPerTopic ~/ 2);
  
  // 3. 生成摘要（OpenClaw 风格）
  final summary = await _generateSummary(oldMessages);
  
  // 4. 保存摘要到话题
  topic.memorySummary = (topic.memorySummary ?? '') + '\n\n' + summary;
  topic.messages = recentMessages;
  
  // 5. 持久化
  await _saveTopic(topic);
}
```

### 5. 话题智能分类（2h）

```dart
enum TopicCategory {
  health,    // 🏥 养生/健康
  finance,   // 💰 理财/股票
  work,      // 💼 工作
  travel,    // ✈️ 旅行
  learning,  // 📚 学习
  life,      // 🏠 生活
  other,     // 📦 其他
}

/// LLM 自动分类
Future<TopicCategory> classifyTopic(ConversationTopic topic) async {
  final prompt = '''
根据话题内容，选择最合适的分类：

标题：${topic.title}
摘要：${topic.summary}
关键词：${topic.keywords.join(', ')}

分类选项：health, finance, work, travel, learning, life, other

只返回分类名称。
''';

  final result = await llmProvider.generate(prompt);
  return TopicCategory.values.firstWhere(
    (c) => c.name == result.trim().toLowerCase(),
    orElse: () => TopicCategory.other,
  );
}
```

### 6. 跨话题检索（2h）

```dart
/// 跨所有话题搜索相关内容
Future<List<TopicSearchResult>> searchAcrossTopics(String query) async {
  final results = <TopicSearchResult>[];
  
  for (final topic in _topics) {
    // 搜索话题标题
    if (topic.title.contains(query)) {
      results.add(TopicSearchResult(
        topic: topic,
        matchType: 'title',
        relevance: 1.0,
      ));
    }
    
    // 搜索话题摘要
    if (topic.summary.contains(query)) {
      results.add(TopicSearchResult(
        topic: topic,
        matchType: 'summary',
        relevance: 0.8,
      ));
    }
    
    // 搜索消息内容
    for (final message in topic.messages) {
      if (message.content.contains(query)) {
        results.add(TopicSearchResult(
          topic: topic,
          matchType: 'message',
          message: message,
          relevance: 0.6,
        ));
      }
    }
  }
  
  // 按相关度排序
  results.sort((a, b) => b.relevance.compareTo(a.relevance));
  return results;
}
```
