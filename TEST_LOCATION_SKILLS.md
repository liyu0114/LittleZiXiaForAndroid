# 小紫霞 v1.0.23 - 位置相关 Skills 测试版

## ✅ 新增功能

### 🎯 6 个位置相关 Skills（使用新的指令执行机制）

1. **local_weather** - 基于当前位置查询天气
   - 自动获取所在城市
   - 使用 Open-Meteo API（免费，无需 API key）
   - 示例："我现在所在城市今天的天气"

2. **current_location** - 获取当前位置信息
   - 显示经纬度、海拔、精度
   - 示例："我现在在哪里"

3. **nearby_restaurants** - 搜索附近的餐厅
   - 基于位置的餐厅搜索
   - 示例："附近有什么好吃的"

4. **nearby_gas_stations** - 搜索附近的加油站
   - 基于位置的加油站搜索
   - 示例："最近的加油站在哪"

5. **nearby_hospitals** - 搜索附近的医院/药店
   - 基于位置的医疗设施搜索
   - 示例："附近有医院吗"

6. **distance_to** - 计算到某地的距离
   - 计算直线距离
   - 示例："我离北京有多远"

---

## 🔧 技术实现

### 新的指令执行流程

```
用户输入："我现在所在城市今天的天气"
    ↓
LLM 识别：skill = local_weather
    ↓
SkillManager 加载：assets/skills/local_weather/SKILL.md
    ↓
MarkdownSkillParser 解析：
    - 提取 Dart 代码块
    - 识别为 DartInstruction
    ↓
MarkdownSkillExecutor 执行：
    - 检测到 Geolocator + open-meteo.com
    - 调用 LocationService 获取位置
    - 调用 Open-Meteo API 查询天气
    ↓
返回结果：
    "🌤️ 多云 ⛅
     🌡️ 温度: 25°C
     💨 风速: 12 km/h
     📍 位置: 39.9042°, 116.4074°"
```

### 关键改进

1. **零代码添加 Skill** - 只需创建 Markdown 文件
2. **动态指令识别** - 自动识别 Dart/HTTP/Bash 指令
3. **智能执行** - 根据代码内容选择正确的执行路径
